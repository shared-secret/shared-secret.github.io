Instructions

  1. A random My public key is generated every time this page is refreshed. It is read only.
  2. Send this key to another party which has their own instance of this page.
  3. Get their public key from them.
  4. Do not refresh this page while the public keys are being swapped.
  5. Paste their public key into the designated field. The shared secret will appear if the key is valid.
  6. The other party must do the same on their end to view the shared secret.

The shared secret can be used as a password to encrypt and decrypt a private message between you and the other party. For example, zip files can be password encrypted and sent as email attachments. Tip: send an encrypted message containing a simpler password.
